# 摄影师预约小程序（photographer）

> 太原本地摄影师中介撮合平台。让用户能找到摄影师，让摄影师能找到活儿。
>
> 核心模式：摄影师入驻 → 用户浏览 → 在线下单 → 微信支付（全款托管）→ 摄影师确认接单 → 拍摄 → 用户确认收片 → 平台扣抽佣后结算。

## 技术栈

- **后端**：FastAPI + SQLAlchemy + SQLite，独立 systemd 服务监听 8001 端口
- **小程序**：Taro 4 + React + TypeScript + Zustand
- **对象存储**：京东云 OSS + 京东云 CDN（与服务器同云内网传输，不占公网带宽）
- **支付**：微信支付 JSAPI（全款托管模式）
