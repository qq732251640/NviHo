# 摄影师入驻协议

> **版本号**: v1.0 · **生效日期**: 2026-05-06
>
> ⚠️ **本协议为初稿,正式上线前须经法律专业人士审核定稿。**

本协议由 [公司全称] (下称"本平台"或"我们")与申请入驻本平台的摄影师(下称"您"或"摄影师")签订。

**在申请入驻前,请您仔细阅读本协议全部内容。提交入驻申请,即视为您已阅读并完全同意本协议及《服务承诺书》各项条款。**

---

## 一、入驻资格

1.1 申请入驻的您应具备以下条件:

- 年满 18 周岁,具备完全民事行为能力
- 拥有真实的摄影作品和拍摄技能
- 拥有合法稳定的工作场地和必要的拍摄器材
- 愿意接受本协议及《服务承诺书》全部条款约束
- 提交真实有效的实名认证信息

1.2 入驻申请时,您需要提交:

- 真实姓名、身份证号、手机号
- 6-12 张代表作(必须是您本人原创)
- 1 段不少于 15 秒的拍摄花絮视频(证明您本人在拍摄现场)
- 摄影领域擅长品类(婚礼/写真/儿童/孕妇/商务等)
- 套餐价格表
- 接单范围(基地城市 + 服务半径)
- 联系电话和微信(平台保密存储,订单成立后才向用户解锁)

1.3 平台运营将在 **3-5 个工作日内** 审核您的申请,不达标的将退回并说明改进方向。

## 二、原创承诺(核心条款)

2.1 您承诺并保证:

- 提交至本平台的所有作品(包括但不限于头像、封面图、作品集、套餐示例图)均为您 **本人原创拍摄**
- 您对所有作品拥有完整、合法的著作权,不存在权属纠纷
- 作品不侵犯任何第三方的知识产权、肖像权、隐私权等合法权益
- 模特、客户已签署或同意您将作品用于平台展示

2.2 **平台对盗图行为的零容忍政策**:

- 平台将通过 EXIF 元数据校验、暗水印追溯、定期反向搜图、用户/同行举报等方式核查作品真实性
- 一经查实有盗图行为,无论是否被申诉:
  - 立即下架所有作品
  - 冻结账号并拉入黑名单
  - 已结算款项可被平台追回(用于赔付被侵权方)
  - 您应向平台支付违约金 **¥5,000-¥50,000**(视情节)
  - 情节严重的,平台保留追究刑事责任的权利

2.3 用户或同行举报盗图查实的,举报方将获得 ¥100 奖励。

## 三、订单接单与服务

3.1 您应在 **24 小时内** 响应每一笔订单(接单或拒单)。

3.2 接单后,您应:

- 提前与用户确认拍摄日期、地点、人数等细节
- 准时到达约定的拍摄地点
- 按套餐约定的拍摄时长和精修张数提供服务
- 在约定的交付时间内通过本平台上传成片预览图和百度云原片链接

3.3 您应通过本平台规定的方式交付成片(详见《服务承诺书》第三条),**严禁绕过平台直接交付**。

3.4 如确实因不可抗力(疾病、自然灾害等)无法履行订单,您应及时通知平台和用户,协商解决方案。

---

## 四、平台抽佣与资金清算

4.1 本平台作为撮合方,从每笔成交订单中按 **约定比例** 收取技术服务费(下称"佣金")。具体比例:

- **新摄影师前 3 个月**: 免佣(为吸引早期入驻摄影师的优惠政策)
- **常规期**: 5%-15% (根据等级和品类不同,具体在套餐管理页明示)

4.2 资金清算流程:

```
用户支付 → 第三方持牌支付机构托管(冻结)
   ↓ 用户确认收片 / 7 天到期 / 申诉裁定
按比例自动拆分 → 摄影师收款方账户(您本人微信/支付宝/银行卡) + 平台账户
```

**款项不进入平台对公账户,由具有央行支付牌照的第三方机构直接结算给您**,完全合规无二清风险。

4.3 您需要在入驻时绑定 **本人实名** 的微信号 / 支付宝账号 / 银行卡作为分账接收方。绑定虚假实名信息导致款项无法到账的,后果由您自行承担。

4.4 款项结算时间:

- 用户主动确认收片: 立即触发分账, **T+1 工作日** 到账
- 7 天自动确认: 第 8 天触发分账, T+1 到账
- 申诉处理中: 暂不结算,仲裁结束后按结果分账

4.5 平台抽佣比例 **以订单创建时的比例为准**,后续平台调整抽佣比例不影响历史订单。

## 五、防绕单条款

5.1 平台严禁您与用户私下交易绕开平台。具体禁止行为:

- 在订单沟通中向用户索要微信号、电话、绕过平台直接交易
- 在作品图片、套餐说明、个人简介中留个人联系方式
- 引导用户取消平台订单后线下成交
- 通过其他平台或渠道与本平台用户重复成交相同订单

5.2 平台对绕单行为的处理:

- 站内 IM 自动检测敏感词("加我微信""V 信"等),多次触发会限流账号
- 用户举报绕单查实的,违约金 **¥3,000-¥20,000**(视订单金额倍数计算)
- 二次绕单立即冻结账号,所有未结算款项扣留作为违约金

5.3 您与用户在订单完成后建立的合法私人联系(用户主动加您微信问候等),不视为绕单。但**与用户复购时,您必须引导其再次通过本平台下单**。

## 六、服务质量与评价

6.1 您应严格遵守《服务承诺书》中的服务标准。

6.2 用户在订单完成后会对您进行 5 维度评分(出片质量 / 准时 / 修图 / 引导 / 服务态度),评分将影响:

- 您的综合评分和评价数(在详情页公开)
- 平台 hot_score 算法(影响您在列表页的排序权重)
- 您是否能晋升 ⭐ 资深摄影师 或 ⭐⭐ 平台优选 等级

6.3 平台运营每月随机抽查 5% 完成订单,人工审核服务质量。连续 3 单评分低于 4.0 或抽查发现明显违反《服务承诺书》的,平台有权暂停您的接单权限,直至整改通过。

## 七、复拍兜底

7.1 如您因以下"重大失误"导致成片质量严重不达标,您应免费为用户复拍 1 次,且退还套餐价差(如有):

- 成片整体虚焦
- 严重欠曝/过曝且无法后期挽救
- 关键瞬间漏拍(如婚礼宣誓、敬酒等)
- 文件丢失或损坏导致无法交付
- 拍摄人员到位但中途离场未完成约定时长

7.2 复拍因您的原因产生的额外费用(交通、场地、模特等)由您承担。

## 八、账号管理

8.1 您可以随时申请下架,但未完成的订单需要继续履行直至完成。

8.2 平台有权基于以下原因冻结或注销您的账号(无需事先通知):

- 违反本协议或《服务承诺书》核心条款
- 提交虚假入驻信息
- 长期(超过 90 天)无任何活跃记录

9.4 用户对作品的使用范围由用户协议约定(个人非商业使用为默认范围)。

10.3 平台不会未经您同意将您的个人信息出售或共享给与本协议无关的第三方。
